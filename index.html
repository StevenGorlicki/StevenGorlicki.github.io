Project 1: Sudoku Solver

In this project I used ML search heuristics to solve sudoku boards. I implemented a custom sudoku board generator, and to solve it I used Forward Checking, Least Constrained Variable, Minimum Remaining Value, and Norvigs Heuristics. This was able to solve a maximum board size of 6x6 efficiently. Created in Python.

Project 2: 2048 Solver

For this project I made a Deep Q-Network to play the game 2048. I also tested other search functions like MTCS and Expectimax heuristics.I achieved an average score of 2048 using Expectimax heuristics, and 512 with DQN. Created using Python / Jupyter Labs.
