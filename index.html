Project 1: Sudoku Solver

Experimenting with ML-based techniques, I created a Sudoku solver and a custom puzzle generator. By integrating Forward Checking, Least Constrained Variable, Minimum Remaining Value, and Norvig's Heuristics, this solution tackles sudoku boards up to size 6x6 with remarkable efficiency. Built entirely with Python.

  • Check it out

Project 2: 2048 Solver

A Deep Q-Network I built to play the puzzle game 2048, while also comparing alternative methods like MTCS and Expectimax Heuristics. Achieved an average score of 2048 using Expectimax, and 512 with the DQN approach. Developed using Python and Jupyter Labs.

Project 3: Bookly App

An app designed for avid readers, Bookly syncs with custom spreadsheets to log completed titles, displays covers using the Google Books API (similar to Kindle's interface), and features custom in-app web searching. Additional functionalities enhance the reading experience and streamline book tracking.
